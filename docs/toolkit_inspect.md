## scarecrow inspect
The `inspect` tool reads a scarecrow pickle file (`.pkl.gz`) generated either by scarecrow seed or scarecrow encode, and returns a summary of its data structure.

```bash
scarecrow inspect --in barcode.BC1.seed.pkl.gz
```
